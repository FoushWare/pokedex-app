# ⚡ Pokédex Explorer

A high-performance, responsive Pokémon browser built with **React 18**, **TypeScript**, and **Tailwind CSS**.

## 🚀 Live Demo
[View on Vercel](https://pokedex-app-nine-rust.vercel.app)

## ✨ Features
- **Dual View Modes**: Switch between traditional **Pagination** and a modern **Load More** list.
- **Detailed Stats**: Comprehensive data on height, weight, types, base stats, and abilities.
- **State Management**: Optimized data fetching and caching with **TanStack Query (v5)**.
- **Performance**: Modular architecture with `React.memo` and `React Suspense` for smooth loading.
- **Error Handling**: Graceful degradation with **Error Boundaries** and retry fallbacks.
- **Responsive Design**: Pixel-perfect layout across Mobile, Tablet, and Desktop.

## 🛠️ Tech Stack
- **Framework**: React 18 (Vite)
- **Language**: TypeScript
- **State/Data**: TanStack Query (React Query)
- **Routing**: React Router v7
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## 📦 Run Locally

1. **Clone the repository**:
   ```bash
   git clone git@github.com:FoushWare/pokedex-app.git
   cd pokedex-app
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the dev server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
